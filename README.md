📚 AI Story Generator — Powered by Generative AI
Welcome to the AI Story Generator, your creative storytelling companion. Turn a simple idea into a full-fledged, beautifully crafted story — all powered by OpenAI’s GPT models. Whether you're a writer, student, or just curious, this app helps you bring your imagination to life through words.

Built with Streamlit, it’s fast, simple, and fun to use.

🌟 What You Can Do
✨ Pick from multiple genres: Fantasy, Sci-Fi, Romance, Mystery, Horror, Adventure, and more

🎭 Choose a tone: Serious, Funny, Dark, Uplifting, Dramatic – you decide the vibe

📏 Set the length: Short (~200 words), Medium (~500), or Long (~1000)

🧠 Smart prompt enhancer: Your basic idea gets a creative boost automatically

📄 Download your story in one click

🔢 Get real-time word count and metrics

🖼️ Browse sample stories for inspiration

📱 Works beautifully on desktop and mobile

🚀 Getting Started
🛠️ Requirements

Python 3.7+

API Key → Generate yours

🔧 Installation

1. Clone this repo
   git clone https://github.com/your-username/ai-story-generator.git
   cd ai-story-generator

2. Install the dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run app.py

4. Open your browser
   Go to http://localhost:8501
   Enter your API key in the sidebar
   Start generating amazing stories!

💡 How It Works

Enter your OpenAI API key
Pick your genre, tone, and story length
Type your story idea (e.g., "A robot discovers emotions")
Hit Generate Story
Download it, share it, or start again!
